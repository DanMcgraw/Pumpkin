# Bedrock Reproduction Matrix

Quick start for the controlled Bedrock join-crash matrix.

```powershell
# 1. Build both binaries
.\scripts\build.ps1

# 2. Copy runtime data into the matrix
.\scripts\setup-runtimes.ps1

# 3. Run all four cells
.\scripts\run-matrix.ps1

# 4. Fill in results/*.md using results/template.md
```

See `plan.md` for the full rationale and interpretation guide.
